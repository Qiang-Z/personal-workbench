## 如何查看历史版本

1. 切换到本 archive 分支
2. 进入对应版本目录（如 `v4.1/`）
3. 直接用浏览器打开该目录下的 `portable/个人工作台.html`

每个版本目录下的 `<version>_升级说明.md` 有详细的版本间升级说明。
