# 个人工作台 v2

一个**纯前端、单文件**的个人效率工作台，覆盖 项目管理 / 科研 / 生活 / 运动 / 金融 五大板块。打开即用，**无需后端、无需数据库**。

## ✨ 功能模块

- 📁 **项目管理**：进行中项目、近期待办与截止、临时任务、按日期的日程日历
- 🔬 **科研**：在投论文、专利、科研项目跟踪
- 🌿 **生活**：读书清单、旅行 & 出行、纪念日 & 生日
- 🏃 **运动**：近 7 天运动、今日运动计划、连续记录、体重 / 测量记录
- 💰 **金融**：基金持仓 / 自选、基金净值记录、本月收支、记一笔收支
- 📌 **今日聚焦**：一屏汇总当天最重要的事项与即将到来的安排

## 🚀 使用方法

直接用浏览器打开本仓库的 `个人工作台v2.html` 即可。所有数据默认保存在**浏览器本地（localStorage）**，不上传任何服务器。

## 💾 数据存储与同步

- 默认数据保存在本机浏览器的 `localStorage`。
- 支持 **GitHub Gist 同步**（在「☁ 同步设置」中配置），方便多设备备份。
- 支持 **本地备份与回滚**，防止数据丢失。

> 提示：清除浏览器数据会清空本地记录，建议定期通过 Gist 或本地备份导出。

## 🌐 部署为在线网页（可选）

如需把它当成一个可访问的在线页面，可在仓库 **Settings → Pages** 中启用 GitHub Pages（源分支 `main`，根目录 `/`），即可获得类似 `https://<用户名>.github.io/personal-workbench/` 的访问地址。
