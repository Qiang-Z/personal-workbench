新建习惯

☁ 同步设置（GitHub Gist）

① 在 GitHub → Settings → Developer settings → Personal access tokens 生成一个有 gist 权限的 Token；
② Token 仅保存在本机浏览器，用于读写你自己的私密 Gist；
③ 开启后：本地改动约 0.8 秒自动上传，页面会每 60 秒自动检查远端更新并无感拉取（同浏览器多标签页也会即时互相同步）；
④ 任意设备用同一 Token + Gist ID 即可同步同一份数据（最后保存方优先）。

🕑 本地备份与回滚

每次改动会自动在浏览器里存一份带时间戳的快照（最多保留 30 份），用于 Gist 同步异常或误删时回滚。备份只存在本机，不上传云端。
